1. Введение

Игра "Клетка" - это аркадная игра, в которой пользователь должен ограничить перемещение объектов в заданной области. Объекты движутся по экрану, и задача игрока заключается в создании замкнутых областей (клеток) вокруг этих объектов с помощью мыши или клавиатуры. Игра заканчивается, когда все объекты будут заключены в клетки.

2. Интерфейс игры

Рабочее пространство игры представляет собой прямоугольник заданного размера. Внутри этого прямоугольника хаотично перемещаются несколько объектов, количество которых задается в начале игры. Объекты могут быть любой формы, но их размер не превышает размер самого широкого символа шрифта Times New Roman 14.

Основные элементы интерфейса:

  • Игровое поле - область, где перемещаются объекты.
  • Панель управления - кнопки для запуска и паузы игры, сброса прогресса и выхода.
  • Счётчик объектов - отображает количество оставшихся незамкнутых объектов.
  • Индикатор состояния - отображает текущее состояние игры (идет игра, пауза, игра завершена).

3. Правила игры

Цель игры - ограничить перемещение каждого объекта в пределах замкнутого прямоугольника, нарисованного с помощью клавиатуры или мыши. Клетки можно рисовать с любого места на игровом поле, и они могут смыкаться.

Правила:

  1. Объекты двигаются хаотично по игровому полю, сталкиваясь с его границами.
  2. Игрок рисует клетки, чтобы ограничить перемещение объектов. Клетки рисуются только по горизонтали и вертикали.
  3. Если объект касается стороны незамкнутого прямоугольника, стороны исчезают, кроме сторон ранее замкнутого прямоугольника.
  4. Игра завершается, когда все объекты будут заключены в клетки.

4. Управление

Управление мышью:

  • Левая кнопка мыши - начать или завершить рисование клетки.
  • Правая кнопка мыши - изменение ориентации рисования(вертикали/горизонтальная).

Управление клавиатурой:

  • Esc - отмена рисования текущей клетки.

5. Советы и хитрости

  • Начинайте рисовать клетки с краев игрового поля, чтобы избежать быстрого столкновения объектов с незамкнутыми сторонами.
  • Используйте паузы для планирования следующего шага.
  • Старайтесь замыкать несколько объектов в одну клетку для ускорения завершения игры.
  • Следите за траекторией движения объектов, чтобы предсказать их перемещение и минимизировать риск касания незамкнутых сторон.

Заключение

Эта справочная система должна помочь пользователям быстро освоить игру и улучшить их игровой опыт, предоставляя всю необходимую информацию об интерфейсе, правилах и управлении игрой "Клетка".
